# Crucible CLI

See the [main README](../../README.md) for full documentation and quick start guide.

This crate provides the command-line interface for Crucible, including chat, MCP server, file processing, and session management.

## Development

```bash
# Run tests
cargo test -p crucible-cli

# Build for release
cargo build -p crucible-cli --release
```
